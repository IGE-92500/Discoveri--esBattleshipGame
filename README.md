## Regras do Jogo

- Cada jogador tem duas grelhas 10x10:
  - a sua frota - o tabuleiro do adversário
- Os navios:
  - são colocados na horizontal ou vertical  
  - não se podem tocar entre si  
- Em cada ronda, o jogador dispara três tiros
- O adversário indica:
  - tiros na água  
  - navios atingidos  
- O jogador regista os resultados no seu tabuleiro de ataque  
- Ganha quem afundar todos os navios do adversário
